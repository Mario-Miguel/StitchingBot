◆ loop()

void loop ( )

Función estándar de arduino que se ejecuta en bucle hasta que el Arduino se desconecta.

Intenta leer una entrada en el SerialMonitor destinado para la comunicación bluetooth. En caso de que reciba una entrada, comprueba qué señal es y ejecuta un comando u otro. En caso de no tener ninguna entrada, intenta ejecutar las acciones que se le ha enviado previamente.
